Computer Doctor Expert System

Description:
Computer Doctor is a simple Expert System developed using Python and the Experta library. The system helps diagnose common computer problems based on user inputs and provides possible solutions.

Features:
-Detects power problems

-Detects display issues

-Detects slow computer performance

-Detects internet connection problems

-Suggests simple troubleshooting solutions

Technologies Used

   Python

   Experta Library

Project Structure


project/

│

├── main.py

├── facts.py

├── rules.py

How It Works
 :
 1-The user answers diagnostic questions.

 2-The system converts answers into facts.

 3-The inference engine evaluates the rules.

 4-Matching rules generate diagnoses and suggestions.

Installation:
 Install Experta library:

 pip install experta

 Run the Project 

 python main.py

Example
 :
Does the computer turn on? yes

Is the display working? yes

Is the computer slow? yes

Does internet work? no

Output: 
Computer is slow.

 Close apps or upgrade RAM.


 Internet problem detected.

 Restart router or reconnect Wi-Fi.
